# Petly PSD to HTML/CSS

This is a static PSD/screenshot-to-HTML conversion based on the provided Petly landing page design.

## Files

- `index.html`
- `styles.css`
- `assets/` sliced image assets from the supplied design screenshot

## How to run

Open `index.html` directly in a browser.

For local development, you can also run:

```bash
python3 -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```

## Notes

- Built with clean HTML and CSS only.
- Responsive layout included.
- Product/gallery/hero images were sliced from the provided screenshot because no separate PSD asset files were supplied.
